## Usage

This dashboard is auto-populated when you run:

```
/ads performance-marketing
```

The `performance-marketing` agent reads your exported campaign data, evaluates all 35+ checks across 6 categories, and fills in this dashboard with real findings, scores, and a prioritized 30-day action plan.

For PDF export of this dashboard, run:
```
/ads report
```
